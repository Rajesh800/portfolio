# Rajesh Thangenapally — Professional Portfolio

![Portfolio Preview](./portfolio-preview.png)

A high-performance, premium portfolio website highlighting 10+ years of experience as a Technical Lead, Java Architect, and AI/ML Engineer. The design is inspired by top-tier Awwwards standards and prioritizes dynamic visual storytelling over static templates.

## ✨ Key Features

- **Immersive Solar System Canvas:** A fully custom, fixed HTML5 background canvas featuring a rotating solar system (sun, planets, moons, asteroid belt, twinkling stars) overlaying the entire site.
- **Glassmorphism & Depth:** Sections are beautifully layered with semi-transparent backgrounds and `backdrop-filter` blurring to allow the canvas graphics to glow through.
- **Custom Cursor Interaction:** A built-in trailing dot cursor that expands contextually on hover across links and buttons.
- **Dynamic Reveal Animations:** Intersection Observers trigger smooth scroll events, floating up cards and typography naturally.
- **Clean Typography System:** Uses the premium humanist font **Plus Jakarta Sans** paired with **Inter** and **JetBrains Mono** to emulate the clean professionalism of LinkedIn.
- **Responsive Architecture:** Fully flexible CSS clamp() functions and grid systems ensure a flawless view on 4K monitors, tablets, and smartphones.

## 🛠 Tech Stack

- **HTML5:** Semantic structure emphasizing clean markup.
- **CSS3:** Advanced modern properties including CSS variables, viewport units (`svh`), responsive `clamp()` typography, flexbox/grid architectures, and hardware-accelerated animations.
- **Vanilla JavaScript (ES6+):** No heavy frameworks (Zero dependencies). Utilizes `IntersectionObserver`, `requestAnimationFrame`, and browser canvas API for maximum performance.
